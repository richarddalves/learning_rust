# Fatorial

Calcula o fatorial de um número utilizando um for para iterar sob os seus antecessores enquando multiplica eles pelo produto final.

## Como executar

```
cargo run
```

O programa pedirá para o usuário inseriro número a ser fatorado e retornará o resultado.

## Exemplo

```
$ cargo run

Insira o número a ser fatorado: 5 
O fatorial de 5 é 120.
```

```
$ cargo run

Insira o número a ser fatorado: 7 
O fatorial de 5 é 5040.
```
